Datenschutzerklärung

Allgemeine Informationen

Diese Datenschutzerklärung beschreibt, wie persönliche Daten in Zusammenhang mit dem persönlichen YouTube-Upload-Tool verarbeitet werden. Dieses Tool wird ausschließlich lokal genutzt und ist nicht für die öffentliche Nutzung gedacht.

Verarbeitung persönlicher Daten

Das Tool verarbeitet keine persönlichen Daten von Dritten. Es greift ausschließlich auf die YouTube Data API zu, um Videos hochzuladen, und verwendet dazu nur die Daten des Kontoinhabers, der die Anwendung autorisiert hat.

Datenweitergabe

Es werden keine Daten an Dritte weitergegeben. Alle API-Interaktionen erfolgen direkt zwischen dem Tool und den YouTube-Servern.

Sicherheit

Die Anwendung speichert keine Daten auf Servern oder in einer Cloud. Alle Vorgänge finden lokal auf dem Rechner des Nutzers statt.
