Persönliches YouTube-Upload-Tool

Über dieses Projekt

Dieses Tool wurde entwickelt, um Videos mithilfe der YouTube Data API auf einen YouTube-Kanal hochzuladen. Es ist für den privaten Gebrauch gedacht und aufgrund der spezifischen Konfiguration kaum für unabhängige Dritte nutzbar.

Datenschutzerklärung

Da dieses Tool nur privat genutzt wird, werden keine Daten von Dritten gesammelt oder gespeichert. Weitere Details finden Sie in unserer Datenschutzerklärung.

Hinweis zur Nutzung

Dieses Tool wurde speziell für persönliche Zwecke erstellt und ist nicht für die allgemeine Nutzung vorgesehen. Aufgrund der individuellen Einstellungen und des lokalen Skripts ist eine unabhängige Nutzung durch Dritte nicht möglich.
