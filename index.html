Persönliches YouTube-Upload-Tool

Über dieses Projekt

Dies ist ein persönliches Tool, das die YouTube Data API verwendet, um Videos auf einen YouTube-Kanal hochzuladen. Es ist kein öffentlich verfügbares Tool und wird ausschließlich für private Zwecke verwendet.

Datenschutzerklärung

Da diese Anwendung nur privat genutzt wird, werden keine Daten von Dritten gesammelt oder gespeichert. Für Details siehe unsere Datenschutzerklärung.

Nutzungsbedingungen

Diese Anwendung wird ausschließlich für private Zwecke verwendet und steht nicht der Öffentlichkeit zur Verfügung. Weitere Informationen finden Sie in unseren Nutzungsbedingungen.

Wichtiger Hinweis

Diese Website dient nur dazu, die Anforderungen von Google für die OAuth-Authentifizierung zu erfüllen. Es handelt sich nicht um ein nutzbares Tool oder einen öffentlichen Service.
